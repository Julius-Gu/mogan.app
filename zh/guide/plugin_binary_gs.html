Ghoscript二进制插件 ​

  • 二进制标识：gs
  • 插件源代码：
    • Codeberg
    • Gitee
    • Github
  • 功能：
    • 将PS/PDF矢量图转换为PDF位图，以在墨干中渲染
    • 将PS矢量图转换为PDF矢量图，以将TeXmacs文档导出为PDF文档

如何安装Ghostscript ​

Windows ​

前往Ghostscript官网下载安装包并按照默认路径安装到系统C盘。

如果用户没有按照默认路径安装，则需要手动将gs.exe所在目录添加到PATH中。

macOS ​

使用brew install ghostscript安装。

Linux ​

使用sudo apt install ghostscript等方式安装。
